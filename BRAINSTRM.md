# Possible Ideas: Focus in LifeStyle
# Expense Tracker (offline, simple, clean)
  Impact: Employers love CRUD + charts + persistence.
  Core Features:
    Add/edit/delete expenses
    Categories
    Total and monthly summaries
    Local persistence (CoreData)
  Stretch Goals:
    Receipt photo capture (no OCR needed)
    Budget alerts

# Personal Habit Tracker (daily routines, streaks, notifications)
Impact: Shows data persistence, notifications, and a clean UI.
  Core Features (doable in 2 weeks):
  Add/edit/delete habits
  Track daily completion
  Show streaks
  Store data with UserDefaults or CoreData
  Simple charts using SwiftUI’s Charts (or your own basic graph)
Stretch Goals:
  Reminders via local notifications
  Widgets
  Custom themes
  
# Fitness Tracker for One Activity (steps, workouts, or timers)
Impact: Looks like a “health app” without huge scope.
  Core Features:
  Choose one:
  Step counter (pedometer API)
  Interval timer (HIIT, Pomodoro-style)
  Simple workout logging
  Stats page
  Clean UI
Stretch Goals:
  Apple Health integration
  Weekly goal tracking
